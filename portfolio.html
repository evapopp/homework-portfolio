News Flash

News Flash is a news aggregator website that uses multiple news APIs to fetch articles for a logged in user. The user can select the sources they want articles to be displayed from and we intend to further the app by creating a filter that allows the user to place parameters on the type of articles they are being shown. The app is currently not deployed but will be soon via heroku.

Day Planner

This app allows the user to input events into their work day planner and save the event via localStorage, for when they re-open the page. By using moment.js the planner gives an updated date at the top of the page, and also color coordinate each timeblock based on the current time.

Check out the repo here

Personal Library API

Using the fetch API function, this site allows users to search for book titles and add them to a personal library. The search results include book title, author, genre, description, and a cover photo. Site includes hamburger menu with menu display on hover, ability to save a title to personal library, and carousel of recommended Library Apps.

Check out the repo here
